          Pourquoi Frython ?
          
Tout ce que vous aimez de Python

          
En français, avec tout le sérieux qu'on peut raisonnablement attendre d'un coq.

        


        

          

            
🎭

            
Projet Humoristique

            
Né d'une idée absurde et exécuté avec une rigueur suspecte. Frython est la preuve que l'absurde peut compiler.

          

          

            
🎓

            
Apprentissage Facile

            
Excellent pour apprendre la programmation en français. Les concepts en langue maternelle, c'est plus naturel.

          

          

            
🔧

            
100% Compatible

            
Chaque bibliothèque Python fonctionne. NumPy, Pandas, Django — tout. Parce que Frython génère du vrai Python.

          

          

            
🐓

            
Mascotte Officielle

            
Le Coq Gaulois veille sur votre code. Une mascotte qui fière, élégante, et légèrement condescendante.

          

          

            
⚡

            
Transpilation Rapide

            
Zéro overhead. Le code est transpilé instantanément avant exécution. Même performance qu'en Python pur.

          

          

            
🌍

            
Bilinguisme Supporté

            
def et déf coexistent. True et Vrai aussi. On ne vous force à rien.
